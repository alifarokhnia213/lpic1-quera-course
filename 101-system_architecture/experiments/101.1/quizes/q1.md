#### 11
**It** stands for (Kernel all Symbols) and is simply a named address inside the linux kernel. a Kernel symbol is the name of function or variable inside the linux kernel. during compilation, each symbol is assigned a memory address. the kernel uses these symbols so that kernel modules can locate and use kernel functions. the file `/proc/kallsyms` is a virtual file that displays the kernel's symbol table. it lists the memory address , type & name of each kernel symbol. this information is mainly used for debugging, kernel development and loading kernel modules. `kallsyms` is in `/proc` because it contains runtime kernel information. it is not in `/sys` since kernel symbols are not kernel objects.

#### 14
**A** Logical volume (LV) is a virtual storage device created by LVM or (Logical Volume Manager). unlike traditional partitions, LVs can be resized, extended or reduced without partitioning the disk.
The directory `/dev/mapper` contains device files for logical volumes & other device mapped storages.it is located in `dev` because LVs can act as block devices that the operating system can mount & use like physical disks and partitions.
